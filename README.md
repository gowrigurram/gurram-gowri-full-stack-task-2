# Full Stack Training Task 2

This repository contains a simple front-end web project with a home page and a registration page.

## Structure

- `Client/`
  - `home.html` — Home page
  - `index.html` — Main landing page
  - `register.html` — Registration page
  - `script.js` — JavaScript logic
  - `style.css` — Styling for the pages

## Usage

1. Open `Client/index.html` in a web browser.
2. Navigate to the registration page and test the form.

## Notes

- This is a static client-side project.
- No server or build tools are required.
